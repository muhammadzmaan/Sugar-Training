Overview

A custom GET endpoint. Returns the $args parameter that is passed to the endpoint method.

Path Variables

Name Description
:data Data string to pass to the endpoint.

Input Parameters

This endpoint does not accept any input parameters.

Result

Name Type Description
__sugar_url String The endpoint path.
data String The data from path variable.

Output Example

{
    "__sugar_url":"v10\/MyCustomApi\/GetAccounts\/MyData",
    "data":"MyData"
}
