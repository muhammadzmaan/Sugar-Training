Overview

Set Activty and its sub activities status to Not Applicable

Summary

This request will set Activty and its sub activities status to Not Applicable

Request Arguments

Name Type Description Required
activity_module String Module name of specific Activity i.e. Tasks True
activity_id String Id of specific Activity True

Request Example

{
    "activity_module": "Meetings",
    "activity_id": "e00ce644-2ce5-11ed-8ea1-0242ac120006",
}

Response

There will be no response
Name Type Description
N/A Undefined Nothing is returned
